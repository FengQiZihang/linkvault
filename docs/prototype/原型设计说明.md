## 2. 原型文件说明

| 文件 | 说明 |
| --- | --- |
| `产品原型.html` | 可交互 HTML 原型，包含全部页面和页面跳转逻辑，直接用浏览器打开 |

页面范围：登录注册、信息完善、首页、手动导入、标签管理、标签筛选、搜索、链接详情、我的、编辑用户资料。

## 3. 原型交互约定

### 3.1 页面层级

- 首页、搜索、标签、我的是 App 底部四个一级页面
- 手动导入、标签筛选、链接详情、编辑用户资料是二级页面，使用左上角返回，不展示底部导航
- 登录注册、信息完善属于账号流程，不展示底部导航

### 3.2 弹出层形态

原型只使用两类弹出层：

- 底部弹出层：用于动作菜单，例如标签管理中的标签操作、链接详情的更多操作
- 居中弹窗：用于输入、选择和确认，例如重命名、合并确认、删除确认、编辑备注、调整标签

破坏性操作必须使用居中确认弹窗，例如删除收藏、删除标签、退出登录、清空输入。

### 3.3 即时选择

头像、标签 chip 这类轻量选择操作采用所见即所得，不额外弹提示。选中态通过高亮、勾选或移除 chip 表达。
